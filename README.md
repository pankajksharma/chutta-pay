ChuttaPay
==========
**Carry change in your phone!!**

###Micro payments for smart as well as not-so-smart phones.###

- Offline payments through SMS.
- Transfer of funds from one person to other.
- Gives a unique code to each  of the  user.
- Just give your code to another user and receive Chutte.
- Limit on maximum amount that can be transferred.
- Once an upper limit on account is received, we can easily transfer funds to the user's bank account.
- Simple GUI for Android for easy usage and navigation.

###Monetization Methods###

- Integration of AdMob in Android Application
- Direct Text Based Ads for not-so-smart phones.
- Small payment (Rs 20) mutes the ads.

###Future Possibilities###

- Localization in Hindi and other Regional Languages.
- Integration with service provider for free SMSes
- Profile Generation and Targeted Ads for each user.
